# ⬡ Nexus Draw — CNN AI Edition

Real-time multiplayer drawing & guessing game with AI bots powered by a
custom-trained Convolutional Neural Network (TensorFlow.js, runs in the browser).
No external AI API key required.

## Quick Start

### Step 1 — Train the CNN (one-time, ~3 minutes)
Open `model_training/nexus_draw_cnn.ipynb` in Jupyter.
Run all cells. The notebook exports the model directly into `nexus-draw/model/`.

### Step 2 — Install & run
    cd nexus-draw
    npm install
    npm start

### Step 3 — Open browser
    http://localhost:3001

The CNN loads automatically. Add AI bots in the lobby, start the game, and draw!

## Deployment

### Railway (free, easiest)
    npm install -g @railway/cli
    cd nexus-draw && railway login && railway init && railway up

### Render
1. Push nexus-draw/ to GitHub
2. New Web Service → Build: npm install → Start: npm start

### VPS + Nginx
    npm install -g pm2
    pm2 start server.js --name nexus-draw
    pm2 save && pm2 startup

Nginx config (add Upgrade headers for Socket.io WebSocket):
    location / {
        proxy_pass http://localhost:3001;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_set_header Host $host;
    }

### Docker
    FROM node:18-alpine
    WORKDIR /app
    COPY package*.json ./
    RUN npm install --production
    COPY . .
    EXPOSE 3001
    CMD ["node", "server.js"]

## AI Bot System

| Bot      | Confidence threshold | Pace   | Personality        |
|----------|----------------------|--------|--------------------|
| SwiftBot | 28%                  | ~5.5 s | Fast, often wrong  |
| SageBot  | 52%                  | ~10 s  | Slow, very accurate|
| ArtsyBot | 38%                  | ~8 s   | Balanced           |

Every 1.6 s each player's browser runs the CNN against the live canvas
(downscaled to 28×28). When a bot's confidence threshold is crossed it
emits `bot-guess` to the server — scored identically to a human guess.

## CNN Categories (AI Mode words)
circle, square, triangle, star, cross, house, tree, arrow, sun, fish

Enable "AI Mode" in room settings to restrict word choices to these 10
shapes so bots can actually compete.
